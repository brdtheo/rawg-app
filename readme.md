
# RAWG app
Video game listing in a simple react native app using the [API](https://rawg.io/apidocs)

## Features
- [x] Plain text search
- [x] Game detail
- [x] Reviews list (search screen & game detail)
- [x] Infinite scroll (from pagination)
<br><br>
- [ ] Search results filters / ordering
- [ ] User accounts
- [ ] Game trailers
